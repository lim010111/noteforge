# Code of Conduct

`noteforge` 프로젝트는 **Contributor Covenant v2.1**을 본 프로젝트의 행동 강령으로 채택합니다. 본 문서는 그 채택 사실과 본 프로젝트에 한정된 운영 사항만을 기록합니다.

## Adopted text

본 프로젝트의 행동 강령 본문은 아래 문서 그대로입니다 (한국어 번역도 동일 사이트에서 제공).

- 영문 원문: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- 한국어 번역: <https://www.contributor-covenant.org/ko/version/2/1/code_of_conduct/>

본 레포에 보관된 본 문서와 위 원문 사이에 차이가 있을 경우 **원문이 우선**합니다.

## Scope

본 강령은 다음 공간에 적용됩니다.

- 본 GitHub 레포지토리의 issues, pull requests, discussions, code review 코멘트
- 본 프로젝트가 운영하는 공개 채널 (추후 추가 시 본 문서에 명시)
- 위 공간 밖이라도, 본 프로젝트를 공식적으로 대표하여 활동하는 경우

## Reporting

행동 강령 관련 우려사항은 비공개로 메인테이너에게 보고해주세요.

- 1순위 — 메인테이너에게 GitHub 직접 연락:
  <https://github.com/lim010111>
- 보고서에는 가능한 범위에서 다음을 포함해주세요.
  - 발생 시점·장소(URL 또는 채널)
  - 관련 인물(아이디)
  - 발생한 내용에 대한 사실 기반 서술
  - 본인이 원하는 대응 방향(있다면)

보고자의 신원과 보고 내용은 비공개로 처리되며, 보고를 이유로 한 보복은 그 자체로 본 강령 위반입니다.

## Enforcement

집행 절차는 위 원문에 명시된 **Enforcement Guidelines**를 따릅니다. 본 프로젝트 단계에서는 메인테이너가 트리아지·결정을 수행합니다. 메인테이너 본인에 대한 보고가 필요한 경우, 본 프로젝트는 v0.x 단계이고 거버넌스가 완비되어 있지 않으므로 GitHub의 신고 도구(<https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam>) 사용을 권장합니다.

## Versioning

- 채택 버전: Contributor Covenant **v2.1**
- 채택일: 2026-05-04 (`v0.8` 릴리스 readiness sweep)
- 본 채택 버전을 갱신할 경우 동일 문서에 변경 일자를 함께 명시합니다.
