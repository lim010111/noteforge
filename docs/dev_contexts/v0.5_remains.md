- [done] 포스트 hero 배경 이미지를 블로그 주인이 수동으로 결정할 수 있게 했다. 개발 서버에서만 `이미지 설정` 버튼을 렌더링하고, 선택 값은 frontmatter `cover`에 `/attachments/<vault-rel>` 또는 `https?://` 형태로 역기록한다. `/attachments/<rel>` 값은 public attachment closure에 포함된 경우만 허용하며, cross-origin POST는 CSRF 가드로 거부한다. 정적 build에는 `/__obpub/cover` middleware와 picker UI가 포함되지 않는다.
